## text

青空文庫で公開されているテキストファイルを元にしています．
ただし，以下は除去しています．

- ヘッダ
- フッタ
- 先頭の空白
- ルビ
- ルビ境界記号``｜``
- 音声中で読み上げられない文字(章番号など)

また，JIS外字指定文字は該当する漢字に置き換えています．
(例: ``※［＃「特のへん＋廴＋聿」、第3水準1-87-71］``→``犍``)
